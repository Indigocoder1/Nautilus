    Class StoryGoalHandler

    A handler class for interacting with all of the major goal systems in Subnautica, which are essential in the game progression. Utilizes the following systems:

    • Story.ItemGoalTrackerCompletes an Story.ItemGoal (or multiple) when an object with the given TechType is picked up, equipped, or crafted through the Mobile Vehicle Bay.
    • Story.BiomeGoalTrackerCompletes a Story.BiomeGoal when the player stays in a given biome for a specified period of time.
    • Story.LocationGoalTrackerCompletes a Story.LocationGoal when the player stays within range of a certain position for a specified period of time.
    • Story.CompoundGoalTrackerCompletes a Story.CompoundGoal when all required "precondition" goals have been completed.
    • Story.OnGoalUnlockTrackerHandles the completion of a goal with the data defined in an Story.OnGoalUnlock object. Defines the unlocked blueprints, signals, items, and achievements that should be unlocked.
    • StoryGoalCustomEventHandlerHandles arbitrary code that is executed after completing a goal, for use in more specific story events.

    RegisterOnGoalUnlockData(string, UnlockBlueprintData[], UnlockSignalData[], UnlockItemData[], Id[])

    Registers a new Story.OnGoalUnlock object for an existing goal. Handles complex actions that occur with the goal's completion.

    Important: Since these are stored in a dictionary, only one Story.OnGoalUnlock object can be added for each specific goal key. Therefore, be careful when adding unlock data to base-game features.
