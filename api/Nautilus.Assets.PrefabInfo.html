    Parameters
    Type Name Description
    string classId The class identifier used for the PrefabIdentifier component whenever applicable.
    string displayName The display name of this Tech Type, can be anything. If null or empty, this will use the language line "{enumName}" instead.
    string description The tooltip displayed when hovered in the PDA, can be anything. If null or empty, this will use the language line "Tooltip_{enumName}" instead.
    string language The language for this entry. Defaults to English.
    bool unlockAtStart Whether this tech type should be unlocked on game start or not. Default to true.
    Assembly techTypeOwner The assembly that owns the created tech type. The name of this assembly will be shown in the PDA.
